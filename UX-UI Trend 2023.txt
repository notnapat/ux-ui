  
  UI => นักออกแบบ หน้าตาเว็บ
  UI Kit == ชุดเก็บ ui ไว้เยอะๆ ไว้ใช้งานต่อ หาดูเยอะๆ แล้วรวบรวม แบบฟรี มาไว้ใช้
  
1. Motion Design == เช่น เมื่อกดปุ่มอะไร แล้ว จะมีตัวการ์ตูนเคลื่อนไหว หรือ แอนิเมชั่น หรือ อยู่เฉยก็จะเคลื่อนไหว
2. Scrollytelling && Parallax scrolling == เว็บเมื่อสกลอเมาส์ลงมา จะมีภาพเคลื่อนไหว สมัยใหม่จะผสมบอกเล่าเรื่องราวไปด้วย
3. New Bolder Minimalism == ดูสมัยใหม่ สะอาด ไม่รก ไม่เยอะ คลีนๆ เล็กน้อย แต่เรียบหรู ดูแพง
4. Happy Colors / Dopamine inducing palettes == เว็บใช้สีเยอะๆ เล่นกับสี
5. Aurora Gradients == ไล่เฉดสี ผสมสี ดูแบบเคลื่อนไหว แต่เป็นภาพนิ่ง , เหมือคลื่นบ้าง
6. glassmorphism == คล้ายๆกระจก เช่น บัตรเครดิต ทับอยู่ สามารถมองเห็นสิ่งที่อยู่ข้างหลังได้


UX => นักประสพการณ์ผู้ใช้งาน

1. Cross-device experience == ทำ ให้ทุกอุปกรณ์ เช่น ios android pc ใช้แอพนั้นๆ มีความรู้สึก หรือได้ประสพการ์ณ เหมือนกัน ทุกแพทฟอร์ม
2. Emotional design == ชักจูก ดึกดูด ผู้คนให้เข้ามาใช้งาน และ ทำให้ติดพัน ให้เข้ามาใช้งานเรื่อยๆ
3. (Hyper) Personalization == ตอบโจทผู้ใช้งาน เช่น tittok เมื่อเราดูคลิปใหน หรือคลิปประเภทเดิม บ่อย นาน กดไลค์บ่อย ก็จะเห็น หรือ เจอคลิปประเภทนั้นเยอะขึ้น หรือบ่อยขึ้น
4. MIcro-interactions == มีปฎิสัมพันกับผู้ใช้ เช่น เมื่อคลิกแล้วมีอะไรขึ้นให้ดู เช่น อนิเมชั่น loading ที่ดูสดใส , หรือ กดไลค์ แล้วมีหัวใจเด้ง
